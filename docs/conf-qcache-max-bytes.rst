Integer, in bytes. The maximum RAM allocated for cached result sets.
Default is 0, meaning disabled. Added in 2.3.1-beta. Refer to `query
cache <qcache.html>`__ for details.

.. code:: programlisting

    qcache_max_bytes = 16777216
