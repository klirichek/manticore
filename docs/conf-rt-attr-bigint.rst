BIGINT attribute declaration. Multi-value (an arbitrary number of
attributes is allowed), optional. Declares a signed 64-bit attribute.
Introduced in version 1.10-beta.

.. rubric:: Example:
   :name: example

.. code:: programlisting

    rt_attr_bigint = guid
