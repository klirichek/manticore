Unsigned integer attribute declaration. Multi-value (an arbitrary number
of attributes is allowed), optional. Declares an unsigned 32-bit
attribute. Introduced in version 1.10-beta.

.. rubric:: Example:
   :name: example

.. code:: programlisting

    rt_attr_uint = gid
