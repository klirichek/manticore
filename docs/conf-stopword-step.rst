Position increment on `stopwords <conf-stopwords.html>`__. Optional,
allowed values are 0 and 1, default is 1. Introduced in version
0.9.9-rc1.

This directive does not affect ``searchd`` in any way, it only affects
``indexer``.

.. rubric:: Example:
   :name: example

.. code:: programlisting

    stopword_step = 1
