SQL user password to use when connecting to
`sql\_host <conf-sql-host.html>`__. Mandatory, no default value. Applies
to SQL source types (``mysql``, ``pgsql``, ``mssql``) only.

.. rubric:: Example:
   :name: example

.. code:: programlisting

    sql_pass = mysecretpassword
